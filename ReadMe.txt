This package include follow function.

Image operation:
1.capture image single from webcaram
2.campare two image
3.send the campare result to another socket

Sound operation:
1.capture sound single from sound card
2.analysis the sound frequency
3.send the result to another socket 

At furture need add follow function:

2.use matlib package realization display the sound single as polt.
